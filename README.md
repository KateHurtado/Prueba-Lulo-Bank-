# Prueba-Lulo-Bank-

Buenas tardes Lulo Bank 
El programa en Python se ha realizado usando google Colab 

1) El programa se debe ejecutar en el orden establecido por la numeracion de cada una de las celdas de ejecucion del google colaboratory, iniciando desde el numero uno, hasta el mayor numero de la celdas. 
2) Se desplegara la informacion de Profiling una vez ejecutando la celda correspondiente.
